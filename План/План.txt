Этап 1: Речь -> текст
    Пользователь говорит в микрофон, программа сохраняет речь в текст.

Этап 2: Текст -> мат. выражение
    Текст речи переводится в мат. выражение с уточнением что нужно сделать (решить уравнение, решить неравенство, СЛУ и т.д.).

Этап 3: Мат. выражение -> решение
    Решение мат. задачи с описанием этапов.

Этап 4: Вывод решения
    Например настольное приложение под Linux, используя синтаксис MarkDown. Также вывод графика, если возможно.


--- Подробно об этапе 3 ---

Уравнения:
    1. Линейные* уравнения
        Пример: 4*x - 5 = x
    2. Квадратные* уравнения
        Пример: 5*x**2 - 8*x + 1 = 0
Неравенства:
    3. Линейные* неравенства [1]
        Пример: 4*x - 5 <= x
    4. Линейные неравенства с 2 неизвестными [3]
        Пример: 3*x + 7 - y >= x + 4*y
    5. Квадратные* неравенства [2,3]
        Пример: 5*x**2 - 8*x + 1 > 0
Системы:
    6. Система линейных уравнений с 2 неизвестными
        Пример: / 2*x - 4*y = 0
                \ 6*x + y   = 10
    7. Система линейных* неравенств с 1 неизвестной (+ возможно квадратных*) [3]
        Пример: / 3*x     >  3 - x
                \ 5*x - 7 <= 10
    8. Система линейных неравенств с 2 неизвестными (только графическое решение) [4]
        Пример: / x + y   >  0
                \ 2*x - y <= 4

* - означает, что заместо x может быть другое выражение, например sin(x). В этом случае решается через замену.
